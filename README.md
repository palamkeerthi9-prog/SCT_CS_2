# SCT_CS_2 - Image Encryption Tool

## Objective

To develop a simple image encryption tool using pixel manipulation techniques.

## Description

This project uses Python and the Pillow library to encrypt an image by modifying its RGB pixel values. The encrypted image can be stored separately, demonstrating basic image security concepts.

## Technologies Used

* Python
* Pillow (PIL)

## Input

Image File: input.jpg

Encryption Key: 50

## Output

Image encrypted successfully!

Saved as: encrypted_image.jpg

## Result

The image was successfully encrypted by modifying pixel values and generating a new encrypted image file.
